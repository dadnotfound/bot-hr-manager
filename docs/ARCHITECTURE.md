## 核心模块

### 1. BotHRManager (主入口)
**职责：**
- 初始化所有监控器
- 协调各模块工作
- 提供统一 API

### 2. TokenMonitor (Token 监控)
**职责：**
- 追踪 Token 使用
- 管理预算
- 触发告警

### 3. PerformanceCalculator (绩效计算)
**职责：**
- 计算效率指标
- 计算质量指标
- 计算忠诚度指标

### 4. ViolationDetector (违规检测)
**职责：**
- 检测违规行为
- 执行处理措施
- 记录违规历史

### 5. Database (数据库操作)
**职责：**
- 封装 SQLite 操作
- 提供 CRUD 接口
- 管理数据库连接

## 数据流

### 会话追踪流程

```
1. Bot 开始会话
   ↓
2. BotHRManager.onSessionStart()
   ↓
3. TokenMonitor.trackUsage(sessionId, 0)
   ↓
4. Bot 执行任务
   ↓
5. Bot 结束会话
   ↓
6. BotHRManager.onSessionEnd()
   ↓
7. TokenMonitor.trackUsage(sessionId, tokens)
   ↓
8. PerformanceCalculator.calculateDailyMetrics()
   ↓
9. ViolationDetector.checkViolations()
   ↓
10. 更新数据库
```

## 数据库设计

### 表关系

```
┌─────────────┐
│    bots     │
│  (Bot 基础) │
└──────┬──────┘
       │
       ├──────────────┬──────────────┬──────────────┐
       │              │              │              │
┌──────▼──────┐ ┌────▼────────┐ ┌──▼─────────┐ ┌──▼────────┐
│token_budgets│ │token_usage  │ │performance │ │violations │
│ (Token 预算) │ │ (使用记录)  │ │  (绩效)    │ │  (违规)   │
└─────────────┘ └─────────────┘ └────────────┘ └───────────┘
```

### 索引策略

- `token_usage(bot_id, timestamp)` - 加速查询
- `performance_metrics(bot_id, date)` - 加速统计
- `violations(bot_id, timestamp)` - 加速历史查询
